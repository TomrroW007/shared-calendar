# Glossary（术语表）

- **Space**：共享空间，成员在同一 Space 内共享日历与协作
- **Status**：状态标记（忙碌、休假、空闲、待定）
- **Visibility**：可见性控制（公开 / 私人仅忙碌 / 仅状态）
- **Proposal**：活动提案，包含多个候选日期
- **Vote**：投票结果（有空 / 没空 / 待定）
- **SSE**：Server-Sent Events，用于服务端向客户端单向推送更新
- **Dashboard**：今日概览，聚合所有 spaces 的当天信息
- **Agenda View**：议程视图，按日期分组的列表式日程展示
- **Space Memo**：空间公告/备忘录
